基础操作

由于环境配置先前就已弄好，这里我们直接从博客配置开始。

新建文件夹，我的路径为F:/blog/Ch3mtr4ils.github.io

初始化hexo： hexo init

hexo常用指令：

​ hexo clean：删除之前生成的文件，若未生成过静态文件，可忽略此命令。
​ hexo generate：生成静态文章，可以用hexo g缩写。
​ hexo deploy：部署文章，可以用hexo d缩写。

安装：npm install hexo-deployer-git --save，可以让我们的静态博客挂到GitHub。

预览网站：hexo c && hexo g && hexo s

部署网站到github：hexo c && hexo g && hexo d

安装主题butterfly：jerryc127/hexo-theme-butterfly: 🦋 A Hexo Theme: Butterfly (github.com)
